**1- What is a relative path relative to?**  
**Answer:** Current working directory.

**2- What does an absolute path start with?**  
**Answer:** It starts with the root directory (such as C:\).

**3- What does `Path('C:/Users') / 'Al'` evaluate to on Windows?**  
**Answer:** `C:/Users/Al`

**4- What does 'C:/Users' / 'Al' evaluate to on Windows?**  
**Answer:** It causes an error because we cannot use forward slash to concatenate strings.

**5- What do the `os.getcwd()` and `os.chdir()` functions do?**  
**Answer:** `os.getcwd()` returns current working directory and `os.chdir()` changes the current directory.

**6- What are the . and .. folders?**  
**Answer:** `.` is the current folder but ``..`` is the parent folder.

**7- In C:\bacon\eggs\spam.txt, which part is the dir name, and which part is the base name?**  
**Answer:** `C:\bacon\eggs` is the dir name but `spam.txt` is the base name.

**8- What are the three “mode” arguments that can be passed to the open() function?**  
**Answer:** `r` for read mode, `w` for write mode and `a` for append mode.

**9- What happens if an existing file is opened in write mode?**  
**Answer:** Its content will be erased and overwritten.

**10 - What is the difference between the `read()` and `readlines()` methods?**  
**Answer:** `read()` returns the file's content as a single string but `readlines()` returns a list of strings, each one is a line from the file's content.

**What data structure does a shelf value resemble?**  
**Answer:** Dictionary value.
